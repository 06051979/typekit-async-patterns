Typekit async loading pattern: Standard async pattern

The standard async pattern for loading the Typekit JS asynchronously. Loading asynchronously prevents the Typekit JS download from blocking the rendering of your page.

Recommendations:

  • Place this snippet at the top of the <head> so the download starts as soon as possible

Advantages:

  • Loads Typekit fonts asynchronously (doesn't block page rendering while it loads)

Disadvantages:

  • More bytes than the standard Typekit snippet
  • Causes a flash of unstyled text (FOUT) in all browsers that can't be controlled or hidden
